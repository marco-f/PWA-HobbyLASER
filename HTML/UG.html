PREFACE

This tool is designed to simplify the process of creating and modifying drawings and images for laser engraving and cutting. You can easily create customized projects, make detailed modifications to your images, and generate the G-code necessary to control a GRBL-based CNC laser.

It arises from the need to manage the CNC laser without having to engage our main computer for long working sessions. The integration with an SBC (Single Board Computer) and a web cam allows us Remote Control and Real-Time Monitoring, giving us the ability to make modifications during the processing phase.

MAIN FEATURES

-- Creating and Modifying Drawings:
HobbyLaser offers the ability to create vector drawings through parametric scripts. You can draw from scratch or import existing images (SVG generated with Inkscape) to modify them according to your needs.

-- Generating G-code: Once your project is completed, HobbyLaser automatically converts the drawing into G-code, ready to be sent to the CNC laser. This process ensures optimal precision and control during cutting and engraving.

-- AI Integration:
Artificial Intelligence, leveraging a phenomenon called 'divergence' (comparison between theoretical and actual), will alert us and put the machine in safety mode, allowing us the opportunity to intervene.

GRAPHICAL INTERFACE

The interface features a drawing area, a script editor, a command line, a log console, interactive rulers (and guidelines), dropdown menus, a server monitor, and a user guide. In the dropdown menu, we find the classic options: Open (hobbyLaser project), Import (.svg, .png, .jpg, .dxf is experimental), Save as (HobbyLaser project), Download (file specific to FS).
The rulers display steps in pixels, but the drawings will be created in millimeters. An idea of the actual dimensions can be obtained through the rendering tool. The drawing area consists of a grid (snap to grid is default), which can be configured through specific commands.
The console area (height adjustable) offers space for any logs, and the server monitor reports the status of the SBC.
The command line is the control tool for HobbyLaser; by entering simple commands, we can access all functionalities. Refer to this guide to discover them.
The scripting area (iavascript) allows us to create the points of SVG paths that will be seen in the drawing area. HobbyLaser is a parametric CAD, which means it is a computer-aided design system that uses parameters to define the geometric characteristics of a model. The parameters can be numerical variables, equations, or geometric relationships, and they allow easy modification of the model by changing the parameter values.

To create models, we will use three tools: polyline, polygon, text.
To combine and manipulate models, we use the Boolean functions union, difference, and intersection; for transformations, we can use the functions translate, scale, resize, mirror, offset, rotate and skew X and Y.

POLYLINE
The 'polyline' tool allows us to create paths of lines and Bezier curves (quadratic by default) using the mouse. A left click, move, and left click will create a line; a left click, drag, and release will create a curve (scroll to zoom), and with shortcuts, we will have the ability to add/remove points, switch Beziers from quadratic to cubic, close the path, and exit the tool.

e.g.
In the command line, type: polyline() (including the parentheses). The tooltip area will show a list of shortcuts: we can start drawing our design by clicking the mouse; closing the segment is not necessary: right-click closes the path and exits the tool.
In the script area, the following will appear: "var _1 = PV['p_1'];" to which we will need to add return _1 (the 'return' will always refer to the variable we want to display); NOTE that the variable names '_num.' are automatic, so if we declare other variables, we must use a different system. At this point, right-click in the script area, then 'exec,' and the SVG path will reappear in the drawing area.

POLYGON
The 'polygon' tool creates regular polygons by specifying the number of sides, the X and Y position of the center, and the radius size.

TEXT
The 'text' tool creates paths corresponding to text strings by specifying the font type, font size, start X and Y positions, and the value of the string.
